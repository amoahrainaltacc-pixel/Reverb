🎵 PREFIX COMMAND SYSTEM:

Music Commands:

.play <song/url>
- Search and play music
- Add songs to queue

.pause
- Pause current song

.resume
- Resume music

.skip
- Skip current song

.stop
- Stop music and clear queue

.queue
- Show current queue

.nowplaying
- Show current song player

.volume <number>
- Change volume

.loop
- Toggle loop mode

.shuffle
- Shuffle queue

.leave
- Disconnect from voice

.lyrics <song>
- Show lyrics

.search <song>
- Search songs


⚙️ BOT COMMANDS:

.help
- Modern interactive help menu

.ping
- Show bot latency

.stats
- Show bot statistics

.uptime
- Show uptime

.invite
- Show invite link

.support
- Show support server
